CVPR 2022 Atmospheric Turbulence Mitigation Challenge (2nd Position)

The atmospheric turbulence mitigation challenge at CVPR 2022 is a part of the Workshop on UG2+ Prize Challenge. The challenge addresses the problem of images acquired over long distances through atmospheric turbulences. It is a challenging task as there is no image domain model and no realistic way of acquiring input-output pairs without using wave-propagation simulation. Our team secured 2nd position in the competition

CVPR 2022 Compressed Video Super Resolution Challenge (Top 12)

The challenge provides LDV 2.0 dataset for performing superresolution on compressed video frames. We proposed the Residual Denoising and Feedback Networks (RDFDBK-Net) for solving the challenge. Our team was listed in Top-12 results. The method was highlighted in a paper published at CVPR workshops 2022.

Deep Convolutional Generative Adversarial Networks

DCGAN uses convolutional and convolutional-transpose layers in the generator and discriminator, respectively. It was proposed by Radford et. al. in the paper Unsupervised Representation Learning with Deep Convolutional Generative Adversarial Networks. The repository provides the code to create varied images of motorbikes from caltech 101 dataset.

Custom Environment for Reinforcement Learning

The repository provides a simple Q-learning implementation for custom environment. There is one hunter and one tiger. The scenario is simple, Tiger has to fetch the meat and run without getting caught. For further simplicity, the hunter remains still. You can change the parameters accordingly but due to crude coding, qtable consumes a lot of memory and space so I have just trained it for 1000 episodes and show the results at every 10th step. You can vary the Grid size as well.

Variational Autoencoders using Tensorflow

This repository provides an implementation of Variational Auto-Encoder on MNIST Dataset. The implementation is based on the description provided in the paper "Auto-Encoding Variational Bayes" by Kingma et al. [https://arxiv.org/pdf/1312.6114.pdf]

Image Denoising using Stacked Autoencoders

Currently, many software and apps have been introduced that reduce noise from images. However, one can train and build their own machine learning model for denoising images. One of the ways to effectively reduce noise from images is the use of stacked autoencoders. A simple implementation of denoising stacked autoencoders is provided in the aforementioned repository

Transposed Convolutional Networks

Transposed convolutional neural networks were designed to upsample the images using learnable parameters. The repository provides an implementation for the transposed convolutional neural networks based on the computations of paper "Fully Convolutional Networks for Semantic Segmentation" published at CVPR 2015. The implementation is based on tensorflow and caltech dataset.

Music Analysis using Librosa

Python code for conducting music analysis using librosa package. The code in this repository represents a tutorial to perform music analysis with librosa package python. This repository is only for educational demo and learning purposes.

Vanilla U-Net Implementation

A Tensorflow implementation of simple U-Net architecture for image segmentation on Oxford Pet Dataset. The code for training and testing U-Net along with the instructions to download the dataset is given in the repository

Generate cartoons uisng GANs

Application of GANs to generate unique cartoo images. The repository provides the MATLAB code for training GANs on cartoonset100k dataset.

Federated vs Centralized Learning

The repository provides both the federated and centralized based learning on MNIST, CIFAR, and FashionMNIST using MLP and CNN based networks. The implementation of federated learning is based on the details provided in the paper Communication-Efficient Learning of Deep Networks from Decentralized Data](https://arxiv.org/abs/1602.05629).

Implementation of Pix2Pix Networks

This repository provides you with the code to implement pix2pix network architecture and train it on Facades Dataset. You can download the dataset using wget command. The instructions are provided in the code.

Segmentation using SegFormer

The code uses SegFormer for Semantic Segmentation on Drone Dataset. The details for the SegFormer can be obtained from the paper: SegFormer: Simple and Efficient Design for Semantic Segmentation with Transformers.

Person Invisibility Demo

Person invisibility demo using OpenCV and Python This code is not perfect as it is based on "Histogram of Oriented GRadients" to detect the person and the fps is slow due to the usage of shallow learning.
